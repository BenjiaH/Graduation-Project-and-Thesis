地球上纬度一度以及经度一度的长度的计算

(2015-05-22 16:04:20)
转载▼
标签：

旅游

一、经纬度：

 

经纬度 是经度与纬度的合称，组成一个坐标系统。又称为地理坐标系统，它是一种利用三度空间的球面来定义地球上的空间的球面坐标系统，能够标示地球上的任何一个位置。

 

二、纬线和纬度：

 

纬线和经线一样是人类为度量方便而假设出来的辅助线，定义为地球表面某点随地球自转所形成的轨迹。

 

纬线指示东西方向。

任何一根纬线都是圆形而且两两平行。

每条纬线长度不同（离赤道越远的纬线越短）。纬线的长度是赤道的周长乘以纬线的纬度的余弦，所以赤道最长，离赤道越远的纬线，周长越短，到了两极就缩为0。

 

纬度 是指某点与地球球心的连线和地球赤道面所成的线面角，其数值在0至90度之间。位于赤道以北的点的纬度叫北纬，记为N；位于赤道以南的点的纬度称南纬，记为S。

赤道的纬度为0°，将地球平分为南半球和北半球。

 

赤道、南回归线、北回归线、南极圈和北极圈是特殊的纬线。

重要的纬线 ： 北极圈（66°33' 38" N）  北回归线（23°26' 22" N）    赤道（0°N)                   南回归线（23° 26' 22" S）南极圈（66°33' 38" S）
三、经线和经度：

 

经线也称子午线，和纬线一样是人类为度量方便而假设出来的辅助线，定义为地球表面连接南北两极的大圆线上的半圆弧。

经线指示南北方向。

任意两根经线的长度相等，相交于南北两极点。每一根经线都有其相对应的数值，称为经度。

 

经度是地球上一个地点离一根被称为本初子午线的南北方向走线以东或以西的度数。本初子午线的经度是0°，地球上其它地点的经度是向东到180°或向西到180°。
不像纬度有赤道作为自然的起点，经度没有自然的起点，做为本初子午线的那条线是人选出来的。英国的制图学家使用经过伦敦格林尼治天文台的子午线作为起点，过去其它国家或人也使用过其它的子午线做起点，比如罗马、哥本哈根、耶路撒冷、圣彼德堡、比萨、巴黎和费城等。在1884年的国际本初子午线大会上格林尼治的子午线被正式定为经度的起点。东经180°即西经180°，约等同于国际换日线，国际换日线的两边，日期相差一日。
经度是指通过某地的经线面与本初子午面所成的二面角。在本初子午线以东的经度叫东经，在本初子午线以西的叫西经。东经用“E”表示，西经用“W”表示。
经度以及纬度的每一度都是被分为60分，每一分被分为60秒。
 
四、纬度一度以及经度一度的长度的计算：
1、纬度一度：
在地球上任何一个地方，纬度每一度之间的距离是一样的。（这个距离是纵向的，南北方向。）
 
每一纬度 长度 = 2 * 圆周率 * R(地球极半径）/ 360度
                       = 2 * 3.1415926 * 6356.9088  / 360
                       = 110.95（千米）
 
纬度每一分 长度 = 110.95 / 60    = 1.85 ( 千米 )
纬度每一秒 长度 =110.95 / 3600 = 1.85 / 60    = 0.0308 （千米） = 30.8 （米 ）
 
2、经度一度：
经度每一度之间的长度，在地球上不同 纬度 的地方，是不一样的。
当然，在同一个纬度的地方，经度每一度的距离是相同的。
在赤道上，经度的每个度大约相当于111km；随着纬度的不同，经度每个度的距离从0km到111km不等。它的距离随纬度的不同而变化，等于111km乘纬度的余弦。不过这个距离还不是相隔一经度的两点之间最短的距离，最短的距离是连接这两点之间的大圆的弧的距离，它比上面所计算出来的距离要小一些。
 
每一经度 长度 =  2 * 圆周率 * R(地球赤道半径）/ 360度 * cosB（当地纬度）
                       = 2 * 3.1415926 * 6377.830 / 360 *cosB
                       = 111.314 *cosB (千米）
 
经度每一分 长度 = 111.314 *cosB / 60      = 1.86*cosB （千米）
经度每一秒 长度 = 111.314 *cosB  / 3600 = 0.0309*cosB （千米） = 30.9*cosB（米）
 
例如，我现在所在地的纬度 为北纬33度，则 cosB = cos33° = 0.83867
每一经度  长度 = 111.314 * cos33 = 93.36 （千米）
 经度每一分   长度 = 93.36 / 60      = 1.56 （千米）
 经度每一秒   长度 = 93.36 / 3600  = 0.0259（千米） = 25.9（米）
 
（在实际工作中，纬度和经度的每一分及每一秒的距离，运用的比较多一点。）
